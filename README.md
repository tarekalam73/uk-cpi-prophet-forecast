# uk-cpi-prophet-forecast
Time series forecasting project using UK CPI data and Meta Prophet in R

## Live Project
[Click here to view the full report] https://uk-cpi-prophet-forecast.netlify.app/

This project analyses UK inflation using Consumer Price Index (CPI) data from the FRED database and applies Meta’s Prophet model to generate forecasts.

## Project contents
- R Markdown report
- HTML output
- UK CPI dataset
- Forecast plots and component analysis

## Methods used
- Data cleaning and preparation in R
- Exploratory time series analysis
- Forecasting using Prophet
- Component decomposition
- Train-test validation

## Summary
The project shows that UK inflation fluctuates around a relatively stable level, with high volatility during the 1970s and more stable behaviour in later decades. Prophet captures the general pattern well, although it struggles with extreme spikes.
